## Description

Sample GraphQL API server with Nestjs

## Installation

1. Duplicate .env.dist file and rename it as .env
2. (optional) Modify environment variables (inside .env)

Then:
```bash
$ docker compose up --build
```

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
